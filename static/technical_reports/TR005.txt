<P>&nbsp;<A NAME="_Toc394895185"></A><FONT SIZE=3><B><I>ABSTRACT</I></B>:
This technical report describes the design, implementation and testing
of a system that improves its returnability characteristics. The system
consists in a pair of control valves, that, added to a modified power steering
system, create a pressure shift on the power steering system when proper
commands are issued.</FONT>
